# Addressbook
A simple single page addressbook app powered by Ember.js

# Requirements
- Node.js
- Ember.js - http://www.ember-cli.com/

# Installation
```
git clone https://github.com/sebastianschnock/addressbook.git
cd addressbook
ember install
ember serve
```
Access at [http://localhost:4200](http://localhost:4200)
